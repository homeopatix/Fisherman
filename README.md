# Fisherman – LOTRO Plugin

![Lua](https://img.shields.io/badge/Lua-Plugin-blue)
![LOTRO](https://img.shields.io/badge/Game-LOTRO-darkgreen)
![Status](https://img.shields.io/badge/status-active-success)
![License](https://img.shields.io/badge/license-Custom-lightgrey)

Fisherman est un plugin **Lua pour The Lord of the Rings Online (LOTRO)**  
Fisherman est un petit plugin pour vous aider à la pêche
Affiche tous les poissons disponible, affiche les objects utilie à acheter, 
affiche les maitres de pêche

---

# 🚀 Fonctionnalités

✔️ Affiche tous les poissons disponible
✔️ Affiche tous les vendeurs d'obets de pêche disponible
✔️ Affiche tous les maître de pêche disponible et leur emplacement dans le jeu

---

# 🎮 Commandes & Raccourcis

| Commande / Touche | Action | 
|--------|------------------|
| Afficher / Masquer l'icône | `/Alt icon` |
| Déplacer l’icône | `ALT` |
| Ouvrir les options | `SHIFT + clic sur l’icône` |

# 📌 Commandes Chat

| Commande | Description | 
|--------|------------------|
| /Fish show	| Montre la fenêtre principale |
| /Fish hide	| Cache la fenêtre principale |
| /Fish options	| Affiche la fenetre d'options|
| /Fish toggle	| Affiche ou cache la fenêtre |

------------------------------------------------------------------------

# 🧩 Installation

1.  Télécharger le ZIP depuis GitHub ou LOTROInterface.
2.  Extraire le dossier.
3.  Copier le dossier `Fisherman` dans :

    Documents\The Lord of the Rings Online\Plugins\Homeopatix

4.  En jeu :
    
    /plugins load fisherman

------------------------------------------------------------------------

# 💾 Données & Synchronisation

Les paramètres sont sauvegardés automatiquement à chaque
connexion/déconnexion.

Les fichiers sont enregistrés dans :

    Documents\The Lord of the Rings Online\PluginData\<Compte>\<Serveur>\<Personnage>

------------------------------------------------------------------------

# 🏗️ Architecture

-   Langage : **Lua**
-   Type : Plugin LOTRO
-   Stockage : Données locales (PluginData)
-   Compatibilité : Toutes versions LOTRO récentes
